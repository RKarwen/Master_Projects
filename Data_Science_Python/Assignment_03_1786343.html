Voraussetzungen für den t-Test¶

1. Normalverteilung¶

Wir knnen davon ausgehen, dass die Daten normalverteilt sind, da der Datensatz sehr gro ist.

2. Homogenität der Varianz¶

Bevor wir den t-Test durchfhren, mssen wir die Homogenitt der Varianz berprfen, da diese bestimmt, ob der t-Test oder der Welch-Test verwendet werden soll.

Levene-Test auf Homogenitt der Varianz zwischen den Gruppen

  • H0 fr Levene-Test: Es besteht Homogenitt der Varianz fr beide Gruppen ("beide Gruppen haben hnliche Varianz")
  • H1 fr Levene-Test: Es gibt keine Homogenitt der Varianz fr beide Gruppen ("beide Gruppen haben unterschiedliche Varianz")
In [20]:

nokids = shopping_data_clean.loc[shopping_data_clean['kids'] == "no_kid", :]

Wir verwerfen H0, da der Levene-Test zeigt, dass keine Varianzhomogenitt vorliegt: p < 0.001.

Daher verwenden wir den Welch-Test anstelle des normalen t-Tests.

T-Test

H0: Kunden mit Kindern kaufen gleich viel wie Kunden ohne Kinder </br> H1: Kunden ohne Kindern kaufen mehr als Kunden mit Kinders

def categories_shopping(df):
    if (df['No._WebPurchases'] > 0 and df['No._CatalogPurchases'] == 0 and df['No._StorePurchases'] == 0):
        return "WebPurchases"
    elif (df['No._WebPurchases'] == 0 and df['No._CatalogPurchases'] > 0 and df['No._StorePurchases'] == 0):
        return "CatalogPurchases"
    elif (df['No._WebPurchases'] == 0 and df['No._CatalogPurchases'] == 0 and df['No._StorePurchases'] > 0):
        return "StorePurchases"
    else :
        return "MultiplePurchases"

shopping_data_clean['type_of_purchase'] = shopping_data_clean.apply(categories_shopping, axis = 1)
